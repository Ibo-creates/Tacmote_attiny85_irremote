# Tacmote_attiny85_irremote
A tiny portable attiny85 programmable ir remote inside a tictac box.

I wanted to use this tictac box I had for a tiny gadget or electronics project since its a really good enclosure.So I choose to build a tiny attiny85 IR blaster since I wanted to control my tv and ac with the same remote instead of having 2 seprate remotes. 

Tacmote works on the attiny85 ic conncected to pushbuttons and several other components including a 120 mah battery and a tp4056 usb c charger
